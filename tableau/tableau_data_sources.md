# Tableau Data Sources

## Purpose

This document maps the processed CSV extracts to the planned Tableau dashboard suite. The extracts are generated by `scripts/build_analytics_database.py` and stored in `data/processed/`.

## Extract Mapping

| Extract | Grain | Primary Dashboard |
| --- | --- | --- |
| `mart_executive_scorecard.csv` | One row per month | Executive Distribution Scorecard |
| `mart_lead_source_quality.csv` | One row per lead source | Lead Source Quality Dashboard |
| `mart_advisor_productivity.csv` | One row per advisor | Advisor Productivity Dashboard |
| `mart_book_growth.csv` | One row per advisor per month | Book Growth and Net New Assets Dashboard |
| `mart_client_retention_risk.csv` | One row per client | Client Segment and Retention Dashboard |
| `mart_compensation_alignment.csv` | One row per advisor per month | Compensation Alignment Dashboard |

## Recommended Tableau Build Order

1. Executive Distribution Scorecard
2. Lead Source Quality Dashboard
3. Advisor Productivity Dashboard
4. Client Segment and Retention Dashboard
5. Compensation Alignment Dashboard

## Dashboard Design Notes

- Use `mart_executive_scorecard.csv` as the top-level leadership view.
- Use source, advisor, team, region, and client segment as primary filters.
- Keep monthly trends visible on executive views.
- Use lead source and advisor ranking views to support business decisions.
- Treat retention risk as a prioritization flag until thresholds are calibrated.
- Use compensation metrics to identify outliers, not to imply causal performance conclusions.

## Tableau Storyline

The dashboard suite should help leaders move through five questions:

1. Are acquisition, conversion, assets, and retention trending in the right direction?
2. Which lead sources produce the best business value?
3. Which advisors are most productive, and where are coaching opportunities?
4. Which client segments require proactive outreach?
5. Are incentives aligned with productivity and asset growth?
